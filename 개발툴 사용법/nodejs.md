npm
---
버전조회
> npm -v

설치된 모듈을 상위 뎁스만 보여준다.
> npm ls --depth=0

모듈 설치 i로 단축 가능
> npm install module

모듈 설치 와 함께 `package.json` 갱신
> npm i module --save-dev

모듈 삭제 r로 단축 가능
> npm remove module

설정 초기화 `package.json` 파일 생성
> npm init

노드를 실행 시킨다 `package.json`파일의 `script.start` 설정값을 실행
> npm start

`package.json`파일의 스크립트를 실행한다.
> npm run script
