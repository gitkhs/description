## node 웹서버 모듈
```
> npm install http-server --save-dev
> npm start
```
**package.json**
```
"scripts": {
  "start": "http-server"
},
"devDependencies": {
  "http-server": "^0.10.0"
}
```

## 윈도우 FTP를 이용한 배포
```
> npm run deploy
```
**package.json**
```
"scripts": {
  "deploy": "ftp -s:deploy.txt"
}
```
**deploy.txt**
```
open hostname
userid
password

put source.txt target.txt
bye
```
